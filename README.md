# Network Programming - Socket Calculator
Java Socket 기반 클라이언트/서버 계산기 프로그램

## Files
- CalcServerEx.java : 서버 프로그램 (ThreadPool 기반)
- CalcClientEx.java : 클라이언트 프로그램 (server_info.dat 설정 파일 사용)
- server_info.dat : 서버 IP 및 포트 설정 파일

## How to Run
1. 서버 실행 : CalcServerEx.java
2. 클라이언트 실행 : CalcClientEx.java
